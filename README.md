#### O que é a Estrela da Vez?
#### Estrela da Vez (EdV) é nosso simulador de fases de grupo da copa do mundo, usando a linguagem Dart, usando CLI!
----------------------------------------------------
#### Pré-requisitos:
1. Um terminal com um ambiente Dart configurado. (Tutorial [aqui!](https://dart.dev/get-dart)).
2. Com o ambiente configurado, copie o diretório usando o comando abaixo:

```
git clone https://github.com/MortadelloDev/EstrelaDaVez
```

3. Após copiar o diretório entre na pasta `EdV`
> [!WARNING]
> Não confundir com `EstrelaDaVez`!!! Entre até a `EdV`
----------------------------------------------------
#### Como usar?
1. Após entrar na pasta `EdV`, use o comando `dart run bin/main.dart`. (se der erro tente usar o comando `dart pub get` e tente novamente!).

2. Após disso o seu terminal deve mostrar essa tabela:
